# Predicting the prices of Flights

The goal of this project is to analyze the flight booking dataset that was downloaded from the Kaggle website from the following link
https://www.kaggle.com/datasets/shubhambathwal/flight-price-prediction and extract useful information. This project answers the questions
of whether Price of flights is influenced by various factors such as Airline, Departure Time, City(to and from) and the timeframe
of ticket booking. The dataset was collected in two parts: one for economy class tickets and another for business class tickets. 
Data source was secondary data and was collected from Ease my trip website. 

## Installation

- Clone this repository (navigate to Project)
- Navigate to readme to understand how to install the dependencies and requirements. Make sure to have an upgraded version of pip installed.
- Install requirements.txt file.

## Usage

- Installation: Follow the installation instructions outlined in the readme to set up the project on your local machine.
- Running the Project: Launch webapp.py. 
- Input: Features to predict Price for a new prediction, or Date of prediction made to view past predctions.
- Output: Predicted price/ past predictions.

## Features

- Connectivity with database with use of PostgreSQL.
- Streamlit GUI.
- Airflow DAG's.
